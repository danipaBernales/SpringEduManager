# SpringEduManager

Proyecto desarrollado con Spring Boot.

## Tecnologías

- Spring Boot
- Spring MVC
- Spring Security
- Spring Data JPA
- H2 Database
- Thymeleaf
- Maven

## Funcionalidades

- Gestión de estudiantes
- Gestión de cursos
- Login con seguridad
- API REST
- Persistencia con JPA

## Ejecutar

mvn spring-boot:run
